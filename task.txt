Задача:

1. Имеется список книг "названия произвольные" характеристика книг Автор, количество экземпляров, цена (количество 10шт)
Список храниться на сервере; 						// произвольно БД или структура

2. имеется клиентский терминал, где пользователь запрашивает название книги;

3. по UDP/IP клиент - программа обращается к серверу с запросом - «Название книги»;

4. Сервер клиенту при совпадении названия, отправляет название книги, автора, количество экземпляров и цену;

5. Связь Сервера и Клиент - программу можно связывать через LocalHost;

6. Предпочтительные языки С, С++;

7. Для клиент - программы упрощенный графический интерфейс обязателен;

8. Сервер и клент разные программы;

9. Доп.требования для передачи по каналу связи: перед передачей в канал (UDP) продублировать сообщение N раз (N = 3,5,7... настраивается),
	При приеме осуществить мажоритарный способ: анализируются N бит из повторно переданных сообщений, если накопилось N/2+1 "1" - считаем, что передалась "1", иначе - "0"

на работу необходимо максимально 7 дней.
